- 作用：分类页面信息与徽章样式。
- 前置参数：
```yaml
---
title: Example Category
description: A description of this category
image:
style:
  background: "#2a9d8f"
  color: "#fff"
---
```
- 字段：
  - `title`：分类名
  - `description`：描述
  - `image`：分类图（可选）
  - `style.background/color`：徽章背景与文字颜色
